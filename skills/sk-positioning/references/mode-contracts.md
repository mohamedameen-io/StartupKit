# sk-positioning Mode Contracts

- Runtime target: 10-15 minutes
- Required outputs:

  - Competitive alternatives snapshot
  - Initial market category recommendation

## standard

- Runtime target: 30-60 minutes
- Required outputs:
  - Full Dunford 5+1 synthesis
  - Positioning statement variants
  - Market category analysis
  - Messaging implications

## deep

- Runtime target: 1-2+ hours
- Required outputs:
  - Everything in `standard`
  - Expanded JTBD analysis
  - Onliness stress test
  - Category narrative and strategic framing depth
