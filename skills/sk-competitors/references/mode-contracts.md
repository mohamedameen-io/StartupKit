# sk-competitors Mode Contracts

## quick

- Runtime target: 10-15 minutes
- Required outputs:
  - Top-3 competitors table
  - One battle card for highest-threat competitor
  - Concise pricing landscape summary
  - One key opportunity + one key risk

## standard

- Runtime target: 30-60 minutes
- Required outputs:
  - Current full competitors workflow output
  - Competitive matrix
  - Battle cards set
  - Pricing landscape
  - Strategic opportunity and risk synthesis

## deep

- Runtime target: 1-2+ hours
- Required outputs:
  - Everything in `standard`
  - Expanded competitor dossiers for highest-threat competitors
  - Moat durability and strategic vulnerability analysis
  - Additional strategic signal depth
