# sk-pitch Mode Contracts

## quick

- Runtime target: 10-15 minutes
- Required outputs:
  - 1-minute elevator pitch
  - 2-minute pitch script
  - Core narrative framing summary

## standard

- Runtime target: 30-60 minutes
- Required outputs:
  - Elevator + 2-minute + 5-minute formats
  - Investor email draft
  - Core Q&A prompt set

## deep

- Runtime target: 1-2+ hours
- Required outputs:
  - Everything in `standard`
  - 10-minute pitch narrative
  - Expanded Q&A prep
  - Pitch scorecard with prioritized refinements
